# Installation

Install Codepurify globally or locally in your project.

## Global Install

```bash
npm install -g @codepurify/cli
```

## Local Install

```bash
npm install --save-dev @codepurify/cli
```

## Verify Installation

```bash
codepurify --version
```

## VS Code Extension

Install the Codepurify extension for syntax highlighting and autocomplete:

1. Open VS Code
2. Search "Codepurify" in Extensions
3. Click Install

The extension provides:
- Syntax highlighting for `{! ... !}` templates
- Autocomplete for context variables
- Template validation
