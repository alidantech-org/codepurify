# Install via VSIX

Install the extension using a `.vsix` file.

## VS Code Installation

1. Open VS Code
2. Press `Ctrl+Shift+X` to open Extensions panel
3. Click "..." menu in the top-right
4. Select "Install from VSIX..."
5. Choose your `.vsix` file

## Supported Editors

- **VS Code** - Full VSIX support
- **Windsurf** - VSIX compatible
- **Cursor** - VSIX compatible

## CLI Alternative

For VS Code, you can install via command line:

```bash
code --install-extension codepurify-1.0.0.vsix
```

## Verification

After installation:
1. Open Extensions panel
2. Search for "Codepurify"
3. Verify extension appears and is enabled

## Update Process

To update:
1. Download new `.vsix` file
2. Repeat installation steps
3. VS Code will prompt to restart
